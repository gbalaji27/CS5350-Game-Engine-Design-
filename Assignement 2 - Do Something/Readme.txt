Implemented two functionalities

1. Keyboard Movement
    Up Arrow - camera moves in the front direction ( x- axis )
    Down Arrow - camera moves in the back direction ( x- axis )
    Left Arrow - camera moves towards left ( y - axis )
    Right Arrow - camera moves towards right ( y - axis )
    PageUp - camera moves above you ( z - axis )
    PageDown - camera moves below you ( z - axis )

2. Skybox changes every time we run the game
